# 01 - 架构与工程配置

## 1.1 项目概述

Paseo 是一个语音控制的 AI 编码助手，移动端 App 使用 **React Native (Expo 54) + TypeScript** 构建，支持 iOS / Android / Web / Electron Desktop 四个平台。

## 1.4 入口流程

```
index.ts
  ├─ polyfillCrypto()         → crypto.randomUUID polyfill
  ├─ polyfillScreenOrientation() → screen.orientation polyfill
  ├─ import "./styles/unistyles" → 注册 unistyles 主题
  └─ import "expo-router/entry"  → 启动 Expo Router
```

## 1.5 Expo 配置要点

**文件**: `app.config.js`

- **应用名称**: Paseo (production) / Paseo Debug (development)
- **包名**: `sh.paseo` / `sh.paseo.debug`
- **方案**: `paseo`
- **横竖屏**: portrait（竖屏锁定）
- **新架构**: `newArchEnabled: true`
- **运行时版本**: appVersion 策略
- **Ota 更新**: expo-updates
- **权限**: RECORD_AUDIO, CAMERA, MODIFY_AUDIO_SETTINGS
- **插件**:
  - expo-router
  - expo-camera（扫码权限描述）
  - expo-splash-screen（深色/浅色启动屏）
  - expo-notifications（通知图标 + 品牌色 #20744A）
  - expo-audio
  - expo-build-properties（minSdkVersion 29, kotlin 2.1.20, cleartext 允许）
- **实验特性**: typedRoutes, reactCompiler, autolinkingModuleResolution

## 1.7 Babel 配置

**文件**: `babel.config.js`
- 使用 `babel-preset-expo`
- 配置 `react-native-reanimated/plugin`

## 1.8 Metro 配置

**文件**: `metro.config.cjs`
- 扩展解析扩展名: `.web.tsx`, `.web.ts`, `.web.jsx`, `.web.js`
- watchFolders 包含 monorepo 根目录

## 1.9 平台门控常量

**文件**: `src/constants/platform.ts`

```typescript
export const isWeb: boolean;      // Platform.OS === "web"
export const isNative: boolean;    // Platform.OS !== "web"
export const isDev: boolean;       // globalThis.__DEV__
export function getIsElectron(): boolean;    // Electron 检测
export function getIsElectronMac(): boolean; // Electron macOS 检测
```

## 1.10 Polyfills

**crypto**: 使用 `expo-crypto` 的 `randomUUID()` 在 RN 中 polyfill `crypto.randomUUID()`
**screen.orientation**: 为 WebKitGTK 桌面运行时 polyfill `screen.orientation` API
**TextEncoder/TextDecoder**: 核心 polyfill（依赖 `install-crypto-polyfills.ts`）

## 1.11 代码规范

- **TypeScript**: 严格模式（tsconfig 从 monorepo 根继承）
- **Lint**: oxlint + oxfmt
- **测试**: vitest（单元测试）+ playwright（E2E/browser）
- **类型检查**: tsgo（自定义工具）
