## 7.1 Button

**文件**: `src/components/ui/button.tsx`

```typescript
interface ButtonProps {
  variant: "default" | "secondary" | "outline" | "ghost" | "destructive";  // default: "secondary"
  size: "xs" | "sm" | "md" | "lg";                                        // default: "md"
  leftIcon: ReactElement | ComponentType | (() => ReactElement) | null;
  trailing: ReactNode;
  style: ViewStyle;
  textStyle: TextStyle;
  disabled: boolean;
  loading: boolean;
}
```

**渲染逻辑**:
- 使用 `Pressable` 包裹
- `loading` 时显示 ActivityIndicator
- 根据 variant 设置背景/边框/文字色
- 根据 size 设置 padding/minHeight
- hover 状态仅在非 ghost 时改变背景

## 7.10 弹窗/Sheet 组件

**AdaptiveModalSheet** (`components/adaptive-modal-sheet.tsx`):
- 自适应弹窗组件: Mobile 用 `IsolatedBottomSheetModal`(@gorhom/bottom-sheet), Desktop 用 `Modal` + 居中卡片
- Props: `visible`, `onClose`, `header(SheetHeader)`, `snapPoints(string[])`, `desktopMaxWidth(number)`, `testID`
- `SheetHeader` 接口:
  ```typescript
  interface SheetHeader {
    title: string;
    subtitle?: ReactNode;
    back?: { onPress: () => void; label?: string };       // 返回按钮
    leading?: ReactNode;                                     // 左侧图标
    actions?: ReactNode;                                     // 右上动作区(搜索/刷新/关闭)
    search?: { onChange: (value: string) => void; resetKey?: string | number; placeholder?: string; autoFocus?: boolean };
  }
  ```
- Desktop 渲染: 半透明黑色 backdrop(`rgba(0,0,0,0.55)`) + 居中卡片(maxWidth 520, maxHeight "85%", 圆角 xl)
  - 顶部栏: title + actions + 可选的 back/leading/search
  - 滚动内容: `BottomSheetScrollView`(scrollsToTop, keyboardShouldPersistTaps)
  - 底部安全区: `getCompactSheetSafeAreaPadding`
  - 关闭: backdrop 点击 / Escape 键 / X 按钮
- Mobile 渲染: `IsolatedBottomSheetModal` + `BottomSheetBackdrop` + snap points + `BottomSheetScrollView`
- ESC 栈管理: 全局 `escStack` 数组, 按 LIFO 顺序处理 Escape 键

## 7.12 Provider / Model 选择组件
