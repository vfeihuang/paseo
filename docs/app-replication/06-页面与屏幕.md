**版本号获取方式**:
```typescript
// src/utils/app-version.ts
export function resolveAppVersion(): string | null {
  const packageVersion = toVersionOrNull(appPackage?.version);     // package.json
  if (packageVersion) return packageVersion;
  const expoVersion = toVersionOrNull(Constants.expoConfig?.version); // expo-config
  if (expoVersion) return expoVersion;
  const manifestVersion = toVersionOrNull(
    (Constants as unknown as { manifest?: { version?: unknown } }).manifest?.version,
  );
  if (manifestVersion) return manifestVersion;
  return null;
}
```
- 优先级: `package.json.version` → `Constants.expoConfig.version` → `Constants.manifest.version` → `null`
- **鸿蒙端**: 无特殊处理。如 `expo-constants` 无法获取，需用 HarmonyOS 的 `BundleManager.getBundleInfoForSelf` 或 `NativePreferences` 自行实现

**交互逻辑**:
- 三个操作按钮（native 顺序）各自触发对应的连接方式:

- `PairLinkModal` 提交成功 → 同上流程
- `handleOpenSettings` → `router.push("/settings")` — 跳转到设置页
