## 6.1 启动页 / Splash Screen

**触发条件**: 启动时 `resolveStartupRoute` 返回 `kind !== "redirect"`

- 全屏背景色（`surface0`）
- 居中显示 `PaseoLogo`（96px，带 shimmer 动画）

- 4 个操作按钮（垂直排列）：
  - **Copy Logs**: `Copy` 图标 + "Copy logs" → `handleCopyLogs`
  - **Open GitHub Issue**: `RotateCw` 图标 + "Open GitHub Issue" → `openGithubIssue`
  - **Docs**: `BookOpen` 图标 + "Docs" → `openDocs`

**交互逻辑**:
- `handleCopyLogs` → `Clipboard.setStringAsync(logPath)` → 复制 daemon 日志路径到剪贴板

- `openDocs` → `openExternalUrl("https://paseo.sh/docs")` → 打开浏览器
