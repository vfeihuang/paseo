## 3.4 启动路由分发

**文件**: `src/app/index.tsx`

```typescript
function Index() {
  const startupRoute = resolveStartupRoute({ ... });
  if (startupRoute.kind === "redirect") {
    return <Redirect href={startupRoute.href} />;
  }
  return <StartupSplashScreen />;
}
```

**`resolveStartupRoute`** 根据以下条件决定跳转:
- `startupBlocker`（启动阻塞状态）
- `hostRegistryStatus`（主机注册状态）
- `hosts` 列表
- `anyOnlineHostServerId`
- `workspaceSelection` + 有效性检查
