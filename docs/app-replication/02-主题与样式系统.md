# 02 - 主题与样式系统

使用 `react-native-unistyles` 实现运行时主题切换。主题注册在 `src/styles/unistyles.ts`，通过 `UnistylesRuntime.setTheme()` 切换。

## 2.4 暗色主题变体

共 5 种暗色变体，定义在 `src/styles/theme.ts`:

| 主题名 | 表面色(surface0) | accent | 特点 |
|--------|-----------------|--------|------|
| dark | #181B1A | #20744A | 默认，青绿调 |
| zinc | #18181b | #e4e4e7 | 中性灰色 |
| midnight | #161820 | #3b6fcf | 蓝色调 |
| claude | #1f1f1e | #d97757 | 暖橙色 |
| ghostty | #282c34 | #89b4fa | 蓝灰调 |

**构建函数**: `buildDarkSemanticColors(tint: DarkThemeConfig)` — 传入表面色配置，返回完整语义色对象。

## 2.9 主题切换

```typescript
export type ThemeName = "light" | "dark" | "zinc" | "midnight" | "claude" | "ghostty";

export const THEME_TO_UNISTYLES: Record<ThemeName, UnistylesThemeKey> = {
  light: "light", dark: "dark", zinc: "darkZinc",
  midnight: "darkMidnight", claude: "darkClaude", ghostty: "darkGhostty",
};

export const THEME_SWATCHES: Record<ThemeName, string> = {
  light:"#ffffff", dark:"#2D8B62", zinc:"#808080",
  midnight:"#4A6BA8", claude:"#D97757", ghostty:"#8caaee",
};
```

**切换逻辑**（in `_layout.tsx ProvidersWrapper`）:

```typescript
if (settings.theme === "auto") {
  UnistylesRuntime.setAdaptiveThemes(true);
} else {
  UnistylesRuntime.setAdaptiveThemes(false);
  UnistylesRuntime.setTheme(THEME_TO_UNISTYLES[settings.theme]);
}
```

## 2.10 外观动态更新

`apply-appearance.ts` 负责根据用户设置动态更新:
- `uiFontFamily` / `monoFontFamily`
- `uiFontSize` / `codeFontSize`
- `syntaxTheme`

通过 `UnistylesRuntime.updateTheme()` 运行时修改 theme 的 fontSize 和 fontFamily。

## 2.11 Unistyles 注册

**文件**: `src/styles/unistyles.ts`

注册所有主题到 unistyles:
- `light` → lightTheme
- `dark` → darkTheme
- `darkZinc` → darkZincTheme
- `darkMidnight` → darkMidnightTheme
- `darkClaude` → darkClaudeTheme
- `darkGhostty` → darkGhosttyTheme
