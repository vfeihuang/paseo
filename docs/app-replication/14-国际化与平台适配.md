# 14 - 国际化与平台适配

## 14.1 国际化系统

**文件**: `src/i18n/`

### 技术栈
- **i18next** v26 — 核心国际化引擎
- **react-i18next** v17 — React 绑定
- **8 种语言**: 英语(en)、中文简体(zh-CN)、西班牙语(es)、法语(fr)、日语(ja)、葡萄牙语(pt-BR)、俄语(ru)、阿拉伯语(ar)

### 配置

**文件**: `src/i18n/i18next.ts`

```typescript
i18next.createInstance({
  resources: {
    en: { translation: enResources },
    "zh-CN": { translation: zhCNResources },
    es: { translation: esResources },
    fr: { translation: frResources },
    ja: { translation: jaResources },
    "pt-BR": { translation: ptBRResources },
    ru: { translation: ruResources },
    ar: { translation: arResources },
  },
  fallbackLng: "en",
  interpolation: { escapeValue: false },
});
```

## 14.2 平台适配策略

### 平台门控常量

**文件**: `src/constants/platform.ts`

```typescript
export const isWeb: boolean;      // 浏览器/Electron
export const isNative: boolean;   // iOS/Android (RN)
export const isDev: boolean;      // 开发模式
```

### 平台文件后缀

通过 Metro 配置解析平台特定文件:
```
Component.web.tsx    → Web 平台使用
Component.native.tsx → Native 平台使用
Component.ios.tsx    → iOS
Component.android.tsx → Android
```

### 平台特定实现列表

| 通用接口 | Web 实现 | Native 实现 |
|---------|---------|------------|
| `audio-engine` | Web Audio API | `expo-two-way-audio` 原生模块 |
| `audio-recorder` | MediaRecorder | expo-audio |
| `dictation-audio-source` | getUserMedia | 原生录音 |
| `scrollbar-style` | CSS scrollbar | 无（原生滚动） |
| `color-scheme` | matchMedia | Appearance API |
| `unistyles-inline-style` | CSSStyleDeclaration | RN StyleSheet |
| `markdown-text` | HTML 渲染 | NSAttributedString |
| `draggable-list` | @dnd-kit | react-native-draggable-flatlist |
| `sortable-inline-list` | @dnd-kit/sortable | 手势驱动 |
| `browser-pane` | iframe | WebView |
| `diff-scroll` | DOM scroll | ScrollView |
| `sidebar-callout-slot` | Portal | Absolute positioned View |
| `terminal-emulator` | xterm.js | WebView 模拟 |

### Electron 桌面端

**检测**: `import { isElectronRuntime } from "@/desktop/host"`

**文件**: `src/desktop/`

| 模块 | 用途 |
|------|------|
| `host.ts` | 桌面桥接接口（dialog/notification/window/browser） |
| `electron/events.ts` | Electron 事件监听 |
| `electron/window.ts` | 窗口控制更新 |
| `settings/desktop-settings.ts` | Desktop 设置持久化 |
| `updates/` | 应用更新管理 |
| `permissions/` | Desktop 权限管理 |
| `components/` | Desktop 专用 UI（PairDeviceModal, PermissionsSection 等） |
| `daemon/` | 内置 daemon 进程管理 |
