# 12 - 设置页面详情

## 12.1 设置导航

**文件**: `src/screens/settings-screen.tsx`

**视图模型**:
```typescript
type SettingsView =
  | { kind: "root" }                                          // 移动端根
  | { kind: "section"; section: SettingsSectionSlug }         // App 设置 section
  | { kind: "host"; serverId: string; section: HostSectionSlug } // 主机设置 section
  | { kind: "projects" }                                      // 项目列表
  | { kind: "project"; projectKey: string };                  // 项目详情
```

**导航交互**:
- **返回工作区**: `handleBackToWorkspace` → `navigateToLastWorkspace()`(优先) 或 `router.replace(buildOpenProjectRoute())`
- **返回设置根**: `handleBackToRoot` → `router.canGoBack()` ? `router.back()` : `router.replace("/settings")`
- **点击 Section**: `handlePress`(section) → `router.push/replace`(目标路由，根据 `isCompactLayout` 选择) → 渲染对应 section 内容
- **点击主机 Section**: 同上，跳转到 `hostSectionRoute(serverId, section)`
- **选择项目设置**: `handleSelectProjects` → `router.push/replace(buildProjectsSettingsRoute())`

## 12.3 General 设置

**UI 布局** — `SettingsSection` 内嵌卡片:

**发送行为**:
- SegmentedControl: `interrupt` / `queue`
- 描述: 打断当前响应 / 排队等待
- 交互: `handleSendBehaviorChange(behavior)` → `updateSettings({ sendBehavior: behavior })`

**语言**:
- Dropdown 选择器（8 种语言）
- 当前: `settings.language`
- 交互: 点击 trigger → 弹出 `DropdownMenuContent`(width 300, side bottom, align end) → 选择语言项 → `handleLanguageChange(language)` → `updateSettings({ language })`

**服务 URL 行为**（Desktop Only）:
- Dropdown: `ask`(询问) / `in-app`(应用内) / `external`(外部浏览器)
- 交互: `handleServiceUrlBehaviorChange(behavior)` → `updateSettings({ serviceUrlBehavior: behavior })`

**终端回滚行数**:
- TextInput（number-pad 键盘）
- 默认值: `settings.terminalScrollbackLines`
- 交互: `handleTerminalScrollbackChangeText(value)` → 数字过滤 → `commitTerminalScrollback`(onBlur) → 解析并验证 → `handleTerminalScrollbackLinesChange(nextValue)` → `updateSettings({ terminalScrollbackLines })`

## 12.4 Appearance 设置

**文件**: `src/screens/settings/appearance/appearance-section.tsx`

**UI 包含**:

- 代码字号滑块
- 语法高亮主题选择
- 实时预览区域

**交互逻辑**:
- 主题选择: 点击色板预览 → `updateSettings({ theme: selectedTheme })` → 实时应用
- UI 字体选择: Dropdown 选择 → `updateSettings({ uiFontFamily })` → 触发 `applyAppearance`
- 等宽字体选择: Dropdown 选择 → `updateSettings({ monoFontFamily })` → 触发 `applyAppearance`
- UI 字号滑块: 滑动 → `updateSettings({ uiFontSize })` → 实时预览更新
- 代码字号滑块: 滑动 → `updateSettings({ codeFontSize })` → 实时预览更新
- 语法高亮主题: Dropdown 选择 → `updateSettings({ syntaxTheme })` → 实时预览更新

**外观应用** (`apply-appearance.ts`):
```typescript
function applyAppearance(config: {
  uiFontFamily: string;
  monoFontFamily: string;
  uiFontSize: number;
  codeFontSize: number;
  syntaxTheme: string;
}): void;
```

## 12.6 Host 设置页

**文件**: `src/screens/settings/host-page.tsx`

### ConnectionsPage

**UI**:
- 当前连接状态指示（`HostStatusDot`）
- 连接方式列表（SSH/本地等）
- 配对信息（二维码 + 配对链接）
- 重连/断开按钮

**交互**:
- **重连**: `handleReconnect` → `connectToDaemon` → 刷新连接状态
- **断开**: `handleDisconnect` → `disconnect` → 更新 UI
- 配对二维码: 通过 `PairDeviceSection` 渲染

### AgentsPage

**UI**:
- Agent 列表（按状态过滤: active/archived/all）
- 每行: Agent 状态点 + 名称 + 时间 + 操作菜单
- 批量管理（全选 / 批量归档 / 批量删除）

**交互**:
- **过滤切换**: `handleFilterChange(status)` → 更新过滤条件 → 重新渲染列表
- **归档单个**: `archiveAgent(agentId)` → 确认 → 归档
- **删除单个**: `deleteAgent(agentId)` → 确认弹窗 → 删除
- **批量操作**: 选中 + `bulkArchive` / `bulkDelete` → 确认 → 执行

### WorkspacesPage

**UI**:
- 工作区列表（名称 + 路径 + 项目名）
- 清理/归档操作

**交互**:
- **清理**: `cleanWorkspace(workspaceId)` → 释放工作区资源
- **归档**: `archiveWorkspace(workspaceId)` → 归档并关闭

### ProvidersPage

**UI**:
- Provider 列表（显示每个 provider 的连接状态）
- 添加/编辑 Provider
- Provider 诊断: `ProviderDiagnosticSheet`
- Provider 选择器目录: `ProviderCatalogList`

**交互**:
- **点击 Provider**: `handleSelectProvider(providerId)` → 打开 `ProviderSettingsHost`(配置 API key/模型/参数)
- **添加 Provider**: `handleAddProvider` → 打开 `ProviderCatalogList` → 选择目录安装
- **诊断按钮**: `handleOpenDiagnostic(providerId)` → 打开 `ProviderDiagnosticSheet`
- **删除 Provider**: `handleRemoveProvider(providerId)` → 确认 → 移除

### UsagePage

**UI**:
- 各 provider 使用量（token 计数）
- 配额剩余（上限/已用）
- 费用统计（预估费用）

**交互**:
- 自动加载 `useProviderUsage` 查询
- 数据来自服务端 provider 统计 API

### TerminalsPage

**UI**:
- 终端配置文件列表（name, command, args, icon）
- 编辑/删除配置文件
- `TerminalProfileEditModal`

**交互**:
- **编辑**: `handleEditProfile(profile)` → 打开 `TerminalProfileEditModal`(desktopMaxWidth 480) → 编辑 Name/Command/Args → `onSave(draft)` → Promise 完成后关闭
- **删除**: `handleRemoveProfile(profileId)` → `confirmDialog` 确认 → 移除

### HostSettingsPage

**UI**:
- 主机标签编辑（TextInput）
- 连接管理（重连/断开）
- 移除主机（确认弹窗）

**交互**:
- **编辑标签**: `handleLabelChange(text)` → `updateHostLabel` → 持久化
- **移除主机**: `handleRemoveHost` → `confirmDialog` 确认 → `removeHost(serverId)` → `handleHostRemoved` → 路由跳转回 /settings

## 12.8 Diagnostics 设置

**UI**:
- "App Diagnostics" 按钮 → 打开 `AppDiagnosticSheet`
- "Test Audio" 按钮 → 播放测试音 → 显示结果

**交互逻辑**:
- **App Diagnostics**: `handleOpenDiagnostic` → `setDiagnosticSheetOpen(true)` → 渲染 `AppDiagnosticSheet`
- **Test Audio**: `handlePlayPress` → `handlePlaybackTest` → `voiceAudioEngine.initialize()` → `voiceAudioEngine.play({ type: "audio/pcm;rate=16000;bits=16", size, arrayBuffer })` → 播放 thinking tone
  - 成功: 结果文字清空
  - 失败: `setPlaybackTestResult(t("playbackFailed", { message }))`

  - `isPlaybackTestRunning` 状态控制防止重复点击

**AppDiagnosticSheet**:
- 应用版本
- 运行环境
- 平台信息
- 音频引擎状态
- 诊断报告文本（代码样式区域 maxHeight 520, 可复制）
