## 12.12 设置弹窗

### AddHostMethodModal
**文件**: `src/components/add-host-method-modal.tsx`
- 使用 `AdaptiveModalSheet` 渲染
- 内容: 三个选项卡片（垂直布局）:
  - **Direct Connection**: Link2 图标 + "Connect directly" 描述 → `onDirectConnection`
  - **Scan QR**: QrCode 图标 + "Scan QR code"（仅 native 平台）→ `onScanQr`
  - **Paste Link**: ClipboardPaste 图标 + "Paste connection link" → `onPasteLink`
- 每项: 18px 图标 + 标题 + 副文本, hover 变 `surface2`, 圆角 xl
- 关闭: backdrop / Escape / X 按钮

### AddHostModal
**文件**: `src/components/add-host-modal.tsx`
- 使用 `AdaptiveModalSheet` 渲染
- 表单字段:
  - **Host**: TextInput（url 键盘）, 必填
  - **Port**: TextInput（number 键盘）, 默认 6767
  - **TLS**: Checkbox 开关（22x22px, accent 色选中）
  - **Password**: TextInput + 可见性切换按钮（eye 图标, 44x44px 热区, surface2 背景）
  - **Advanced**: 可折叠区域 → URI 输入框
- 按钮: Cancel + Connect（填表状态切换）
- 交互: 实时验证 host/port → `buildConnectionUriFromDraft` 构建连接 URI → `connectToDaemon` 测试连接 → 成功后调用 `onSaved({ profile, serverId, hostname, isNewHost })`
- 成功回调: 调用者通常在此跳转路由（WelcomeScreen: `finishOnboarding()` → 项目首页；SettingsScreen: 返回并刷新主机列表）
- 错误: 移动端 Alert / 桌面端 inline 红色文字（`theme.colors.destructive`）

### PairLinkModal
**文件**: `src/components/pair-link-modal.tsx`
- 使用 `AdaptiveModalSheet` 渲染
- 表单: Offer URL TextInput（自动聚焦, surface2 背景, 圆角 lg）
- 验证: 检测 `#offer=` URL fragment
- 按钮: Cancel + Pair
- 交互: 解析 offer → `connectToDaemon` 测试 → `upsertDaemonFromOfferUrl` → `onSaved`

### TerminalProfileEditModal
**文件**: `src/screens/settings/terminal-profile-edit-modal.tsx`
- 使用 `AdaptiveModalSheet`, `desktopMaxWidth: 480`
- 表单字段:
  - **Name**: 必填, 自动聚焦, 回车跳转到 Command
  - **Command**: 必填, 回车跳转到 Args
  - **Args**: 可选, 带 hint 文字
- 按钮: Cancel + Save
- 交互: 逐字段验证 → `onSave(draft)` → Promise 完成后关闭
- 错误: 行内红色文字（`theme.colors.palette.red[300]`）

### ProviderDiagnosticSheet
**文件**: `src/components/provider-diagnostic-sheet.tsx`
- 使用 `AdaptiveModalSheet`, header 含 search + refresh + copy 动作
- 内容:
  - Discovered models 列表（自动发现）
  - Custom models 列表（用户自定义）
  - 每行: 模型名称 + 删除按钮
- 嵌套子 sheet: `AddCustomModelSubSheet`, `DiagnosticSubSheet`
- 交互: 搜索过滤, 自动 10 秒刷新, 模型增删, 诊断报告复制

### AppDiagnosticSheet
**文件**: `src/components/app-diagnostic-sheet.tsx`
- 使用 `AdaptiveModalSheet`
- 内容:
  - Client diagnostics
  - Desktop diagnostics（桌面端）
  - Host diagnostics（每个已连接主机）
- 进度: 状态圆点（success/destructive）
- 输出: 代码样式区域（maxHeight 520）, 可复制
- 交互: 打开时自动运行诊断, loading spinner, 手动刷新
