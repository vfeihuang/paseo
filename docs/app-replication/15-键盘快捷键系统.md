# 15 - 键盘快捷键系统

## 15.1 架构

**文件**: `src/keyboard/`

- `keyboard-shortcuts.ts` — 所有快捷键定义（约 60+ 绑定）
- `keyboard-action-dispatcher.ts` — 动作注册/分发系统
- `shortcut-string.ts` — 和弦解析和组合键处理
- `focus-scope.ts` — 快捷键作用域管理
- `route-shortcut.ts` — 路由级快捷键

## 15.3 Action Dispatcher

**文件**: `src/keyboard/keyboard-action-dispatcher.ts`

```typescript
// 全局分发器，支持作用域
const keyboardActionDispatcher = {
  dispatch(action: KeyboardAction): void;
  registerHandler(scope: string, handler: ActionHandler): () => void;
  setActiveScopes(scopes: string[]): void;
};

interface KeyboardAction {
  id: string;
  scope: string;
  payload?: unknown;
}
```
